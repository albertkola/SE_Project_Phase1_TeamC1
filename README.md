# Project Details

## Project Title: Intercity Carpooling System with Pickup Points in Albania (Hop In)

## Problem Statement:
In Albania, intercity travel is often expensive, inefficient, and poorly coordinated. Many people travelling between cities rely on informal arrangements or costly transport options, while drivers frequently travel with empty seats. There is no dedicated digital platform that connects intercity drivers and passengers, particularly one that supports flexible, location-specific pickup points suited to the Albanian context.

## Proposed Solution:
We propose a web-based carpooling platform that connects drivers and passengers travelling between Albanian cities. Drivers can post trips with defined pickup points, available seats, and pricing. Passengers can search, filter, and book seats on matching trips, selecting their preferred pickup location. The system streamlines intercity travel by making it more affordable, organized, and accessible for everyday users.

## Project Scope:

- **Aim:** To design and develop a functional intercity carpooling system for the Albanian users, enabling efficient seat-sharing between cities through a structured, user-friendly platform with pickup point management.

- **Objectives:**
  1. Implement a user registration and authentication system supporting both driver and passenger roles.
  2. Enable drivers to create, manage, and cancel intercity trips with custom pickup points.
  3. Allow passengers to search, filter, and book available trips based on route, date, time, and pickup preference.
  4. Build a booking management system with seat tracking, confirmation, and cancellation support.
  5. Develop a ratings and reviews system to build trust between drivers and passengers.
  6. Provide an admin panel for user management, trip monitoring, and platform oversight.
  7. Ensure the system is secure, responsive, and accessible across web browsers and mobile devices.

## Application Description:

**Target Users:**
- **Drivers** — individuals travelling intercity who want to share costs by offering seats to passengers.
- **Passengers** — individuals seeking affordable, flexible intercity travel.
- **Administrators** — platform managers who oversee users, trips, and system activity.

**Key Features:**
- User registration and login with driver/passenger role selection
- Trip creation with departure city, destination, date, time, price, seats, and pickup points
- Trip search and filtering by route, date, price, available seats, and pickup point
- Seat booking with driver approval, automatic seat reduction, and cancellation support
- In-app notifications for booking confirmations, rejections, and trip reminders
- Ratings and reviews for both drivers and passengers after trip completion
- Trip history and booking records accessible from the user account
- Admin dashboard for managing users, trips, bookings, and viewing platform statistics
